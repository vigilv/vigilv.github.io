Projects

This page lists some of my present and past projects.

Handwritten Digit Recognition Using a Spiking Neuron Model

  • This is the current project that I am working on.
  • Biologically feasible neuron models such as the Mihalas-Niebur neuron model provide a convenient way of modeling the different behaviors of a neuron. This project explores a digital neuron model with a conductance based synapse to accurately classify handwritten digits.
  • I will upload the files to GitHub once it is ready.

High Resolution Angle Sensitive Imaging

  • This work was part of my Ph.D. thesis.
  • The pixel design in this work makes use of the trade-off between angular and spatial resolution to implement an angle sensitive image sensor that could be used for practical applications such as depth estimation and auto-refocusing.
  • This work is yet to be published.

Spiking Neuron Model

  • This work implements a biologically feasible spiking neuron model (Mihalas-Niebur neuron model) in Python.

Hybrid Polarization-Incident Light Angle Detection

  • This work aims to make use of the polarization sensitivity of sub-wavelength gratings to detect local polarization as well as incident light angles.

Track-and-Tune Angle Detection

  • This work was part of my Ph.D and it aims to design an image sensor for detecting local incident angle of light. This is a hybrid method that combines the diffraction based Talbot effect pixels and metal shading based quadrature pixel cluster pixels.
  • Paper 1
  • Paper 2

Design of a Speech Analysis-Synthesis System

  • This project explores the ESL design methodology to co-design a speech analysis-synthesis system using MATLAB and SystemC. The design is done for minimum phase systems.
  • Code
  • Report

Design of USB 3.0 Physical Layer

  • This project deals with the digital design of the USB 3.0 physical layer. The physical layer is mainly concerned with the actual transfer of data between host and the device. It consists of two main modules, the transmitter and the receiver.
